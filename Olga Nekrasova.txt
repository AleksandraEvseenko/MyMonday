1. Открыть GitBush
2. ls проверяем содержимое директории
3. cd "название папки" *переходим в необходимую папку 
4. git status *проверяем если ли что на commit. в локальном репозитории. новые файлы в папке помечаются зеленым цветом 
5. git add . *добавляем новый файл в локальном репозитории
6. git status *новый файл в локальном репозитории изменил цвет на зеленый 
7. git commit -m "комментарий" *подтверждает изменения в локальном репозитории
8. git push *отправляем изменения на сервер
